# Product Specifications

This directory contains specifications for Backpack components and features.

## Specifications

| Document | Description |
|----------|-------------|
| [Accessibility Strategy](./accessibility-strategy.md) | WCAG compliance approach and testing strategy |
| [BpkButton](./bpk-button.md) | Complete component spec — props, rendering, SCSS, tests, accessibility |

## Component Documentation

Individual component documentation lives alongside the source code:
- Component READMEs: `packages/bpk-component-{name}/README.md`
- Migration guides: `packages/bpk-component-{name}/docs/`

## Adding New Specs

When proposing a new component or significant feature:
1. Create a spec document in this directory
2. Include: use cases, props API, accessibility requirements, visual examples
3. Reference the relevant Figma design
4. Get design team approval before implementation
