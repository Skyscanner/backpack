BACKPACK SASS PATTERNS - Modern Sass API Reference
====================================================

This reference documents the Sass patterns used in Backpack components.

CRITICAL: Backpack uses the modern Sass API (@use/@forward).
NEVER use @import - it is deprecated and will cause build errors.

COMPILER: sass-embedded v1.90 (Dart Sass)

TOKEN IMPORT PATTERN
---------------------

Every SCSS file starts with:

  @use '../../bpk-mixins/tokens';
  @use '../../bpk-mixins/typography';

The path depth depends on package location. All component packages
are at the same level, so the relative path is always ../../bpk-mixins/.

TOKEN ARCHITECTURE
-------------------

Source chain:
  @skyscanner/bpk-foundations-web/tokens/base.default
    -> bpk-mixins/_tokens.scss (@forward)
      -> Component SCSS files (@use)

bpk-mixins/_tokens.scss contains:
  @forward '@skyscanner/bpk-foundations-web/tokens/base.default';

This means all foundation tokens are available as tokens.* in components.

COLOR TOKENS (Variable-based)
------------------------------

Access as: tokens.$bpk-{semantic-name}-day

Text colors:
  tokens.$bpk-text-primary-day
  tokens.$bpk-text-secondary-day
  tokens.$bpk-text-disabled-day
  tokens.$bpk-text-on-dark-day
  tokens.$bpk-text-on-light-day

Background/Canvas:
  tokens.$bpk-canvas-day
  tokens.$bpk-canvas-contrast-day
  tokens.$bpk-surface-highlight-day

Brand:
  tokens.$bpk-core-primary-day
  tokens.$bpk-core-accent-day

Border:
  tokens.$bpk-line-day
  tokens.$bpk-line-on-dark-day

Static:
  tokens.$bpk-border-size-sm
  tokens.$bpk-border-size-lg

SPACING TOKENS (Function-based)
---------------------------------

Access as: tokens.bpk-spacing-{size}()  (NOTE: these are functions, use parentheses)

  tokens.bpk-spacing-none()    // 0
  tokens.bpk-spacing-sm()      // 0.5rem (8px)
  tokens.bpk-spacing-base()    // 1rem (16px)
  tokens.bpk-spacing-md()      // 1.25rem (20px)
  tokens.bpk-spacing-lg()      // 1.5rem (24px)
  tokens.bpk-spacing-xl()      // 2rem (32px)

Usage:
  padding: tokens.bpk-spacing-base();
  margin: tokens.bpk-spacing-sm() tokens.bpk-spacing-base();
  gap: tokens.bpk-spacing-sm();

TYPOGRAPHY MIXINS
------------------

Import: @use '../../bpk-mixins/typography';

Size-based (maps to font-size + line-height):
  @include typography.bpk-text-xs;
  @include typography.bpk-text-sm;
  @include typography.bpk-text-base;
  @include typography.bpk-text-lg;
  @include typography.bpk-text-xl;
  @include typography.bpk-text-xxl;
  @include typography.bpk-text-xxxl;

Semantic (preferred - includes weight and other properties):
  @include typography.bpk-body-default;
  @include typography.bpk-body-long-form;
  @include typography.bpk-caption;
  @include typography.bpk-footnote;
  @include typography.bpk-heading-1;
  @include typography.bpk-heading-2;
  @include typography.bpk-heading-3;
  @include typography.bpk-heading-4;
  @include typography.bpk-heading-5;
  @include typography.bpk-label-1;
  @include typography.bpk-label-2;
  @include typography.bpk-subheading;

Link:
  @include typography.bpk-link;
  @include typography.bpk-link-underlined;

BEM NAMING IN SCSS
--------------------

.bpk-component-name {
  // Block styles
  @include typography.bpk-body-default;
  padding: tokens.bpk-spacing-base();
  color: tokens.$bpk-text-primary-day;

  &__element {
    // Element styles
    margin-bottom: tokens.bpk-spacing-sm();
  }

  &--modifier {
    // Modifier styles
    color: tokens.$bpk-core-primary-day;
  }
}

CSS MODULES
-----------

File naming: BpkComponent.module.scss

In TypeScript:
  import styles from './BpkComponent.module.scss';
  <div className={styles['bpk-component-name']} />

The .module.scss extension enables CSS Modules scoping.
Class names in the SCSS follow BEM but are locally scoped at build time.

COMPLETE EXAMPLE
-----------------

@use '../../bpk-mixins/tokens';
@use '../../bpk-mixins/typography';

.bpk-card {
  display: flex;
  flex-direction: column;
  padding: tokens.bpk-spacing-base();
  background-color: tokens.$bpk-canvas-day;
  border-radius: tokens.bpk-border-radius-md();
  border: tokens.$bpk-border-size-sm solid tokens.$bpk-line-day;
  box-shadow: tokens.bpk-box-shadow-sm();

  &__title {
    @include typography.bpk-heading-3;
    color: tokens.$bpk-text-primary-day;
    margin-bottom: tokens.bpk-spacing-sm();
  }

  &__body {
    @include typography.bpk-body-default;
    color: tokens.$bpk-text-secondary-day;
  }

  &--highlighted {
    border-color: tokens.$bpk-core-primary-day;
  }
}
