# bpk-component-comparison-table

> Backpack components for building comparison UIs.

## Installation

Check the main [Readme](https://github.com/skyscanner/backpack#usage) for a complete installation guide.

## Usage

```tsx
import BpkComparisonTray, {
  BpkCompareModal,
} from '@skyscanner/backpack-web/bpk-component-comparison-table';
```

## Components

### BpkComparisonTray

A tray for collecting items to compare. See [BpkComparisonTray/README.md](src/BpkComparisonTray/README.md).

### BpkCompareModal

A modal for comparing up to 3 items side by side. See [BpkCompareModal/README.md](src/BpkCompareModal/README.md).
